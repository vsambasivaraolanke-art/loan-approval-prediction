**Loan Approval Prediction**



**Project Overview**



This project develops a Machine Learning classification model to predict loan approval outcomes using structured applicant and financial datasets.



The project demonstrates data preprocessing, missing value handling, feature engineering, model building, and evaluation workflows.



&#x20;**Technologies Used**



\* Python

\* Pandas

\* NumPy

\* Matplotlib

\* Seaborn

\* Scikit-Learn



**Algorithm Used**



\* Random Forest Classifier



**Workflow**



1\. Data Loading

2\. Data Exploration

3\. Missing Value Treatment

4\. Feature Encoding

5\. Train-Test Split

6\. Model Training

7\. Prediction

8\. Performance Evaluation



**Evaluation Metrics**



\* Accuracy Score

\* Confusion Matrix

\* Classification Report



&#x20;**Business Goal**



Build a predictive system to support loan approval decision making using applicant information.



**Results**



Successfully developed a classification model using Random Forest with analytical preprocessing and evaluation techniques.
